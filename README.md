#  Wordle Game and Auto Guess in Telegram Bot## Features* Start a new game in anytime* You can play online with your friends or play it yourself* No limits on the number of guesses* When you hit the answer, it will automatically send you an online definition of Cambridge Dictionary* The word library is similar to that in wordle* Auto guess based on the algorithm ## Screenshots of gameplay###Commands#### Start a new game![](gameplay.png)#### Heck - Auto guess![](heck.png)## Guessing Algorithm## FutureThe target word is stored on the google spreadsheet, which may cause problems when the game is conducted in two or more chat rooms.
